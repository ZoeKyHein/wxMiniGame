# 音频文件说明

请将以下音频文件放入此文件夹：

## 必需文件

- `bgm.mp3` - 背景音乐（循环播放）
- `shoot.mp3` - 射击音效（短促）
- `explosion.mp3` - 爆炸/击中音效
- `levelup.mp3` - 升级提示音
- `win.mp3` - 胜利音效
- `lose.mp3` - 失败音效

## 推荐工具

可以使用 [BFXR](https://www.bfxr.net/) 在线生成 8-bit 风格音效，非常适合本游戏的画风。

## 格式要求

- 推荐使用 MP3 格式（兼容性最好）
- AAC 格式也可以使用
- 文件大小尽量控制在 100KB 以内（音效文件）

## 注意事项

- 如果音频文件不存在，游戏仍可正常运行，只是没有声音
- 背景音乐会自动循环播放
- 音效在播放前会自动重置，确保可以重复播放
