## 📋 发布前检查

### 1. 真机测试（必须）
- [ ] 在真机上测试游戏（至少 5 分钟）
- [ ] 检查手机发热情况
- [ ] 检查帧率是否稳定（60 FPS）
- [ ] 测试摇杆手感
- [ ] 测试按钮大小是否合适
- [ ] 测试音频播放是否正常

### 2. 数值平衡调整
- [ ] 测试前 3 分钟是否太无聊（调整怪物数量/速度）
- [ ] 测试 Boss 是否太难/太简单（调整血量）
- [ ] 测试金币产出是否合理（每局能买 1-2 个天赋）
- [ ] 测试天赋价格是否合理

### 3. 准备上线素材
- [ ] 游戏图标（512x512 px）
- [ ] 游戏简介（一句话）
- [ ] 选择游戏类目（游戏 - 休闲/动作）
- [ ] 准备宣传图（5:4 比例，用于分享）

### 4. 音频文件
- [ ] `audio/bgm.mp3` - 背景音乐
- [ ] `audio/shoot.mp3` - 射击音效
- [ ] `audio/explosion.mp3` - 爆炸音效
- [ ] `audio/levelup.mp3` - 升级音效
- [ ] `audio/win.mp3` - 胜利音效
- [ ] `audio/lose.mp3` - 失败音效

### 5. 配置文件检查
- [ ] `game.json` - 确保 `deviceOrientation` 是 `landscape`
- [ ] `project.config.json` - 设置正确的 `appid`
- [ ] 检查所有资源路径是否正确

### 6. 代码检查
- [ ] 移除所有 `console.log`（生产环境）
- [ ] 检查是否有未使用的代码
- [ ] 确保错误处理完善

## 🚀 发布步骤

1. **上传代码**
   - 在微信开发者工具点击"上传"
   - 填写版本号和项目备注

2. **提交审核**
   - 登录微信公众平台（小游戏后台）
   - 提交审核
   - 填写游戏信息、上传素材

3. **等待审核**
   - 通常 1-3 个工作日
   - 关注审核反馈

4. **发布上线**
   - 审核通过后，点击"发布"
   - 游戏正式上线！
